# ✨ GestureVision: Advanced Hand Tracking System

>  A real-time AI-powered hand tracking and gesture recognition experience built using **MediaPipe**, **JavaScript**, **HTML5 Canvas**, and **Web Audio API**.

---

## 🎯 Project Overview

GestureVision is an interactive computer vision project that transforms hand movements into immersive visual and audio experiences.

Using real-time hand landmark detection, the system recognizes gestures, tracks finger movements, generates dynamic visual effects, and responds with interactive animations and sound feedback.

This project demonstrates the practical application of:

* 🤖 Computer Vision
* ✋ Hand Tracking
* 🎮 Human-Computer Interaction
* 🎨 Interactive Graphics
* 🔊 Audio Processing
* ⚡ Real-Time Rendering

---

## 🌟 Key Features

### ✋ Real-Time Hand Tracking

* Detects and tracks hands instantly using MediaPipe Hands
* Supports tracking of up to **2 hands simultaneously**
* Accurate finger landmark detection

### 👆 Gesture Recognition

* Pinch gesture detection
* Open-hand recognition
* Fist recognition
* Dynamic hand spread analysis

### 🎨 Advanced Visual Effects

* Neon-style rendering
* Particle generation from fingertips
* Interactive shockwave effects
* Dynamic motion trails
* Cross-hand interaction visuals
* Lightning-inspired visual connections

### 🌈 Multiple Visual Themes

Switch between immersive themes:

* 🌈 Rainbow
* 🤖 Cyberpunk
* 🌋 Lava
* 🌊 Ocean
* 🌌 Galaxy

### 🔊 Audio-Reactive Experience

* Real-time sound feedback
* Dynamic audio response based on hand interactions
* Gesture-triggered effects

### 📊 Live Performance Dashboard

Displays:

* Hands detected
* Current FPS
* Gesture status
* Hand spread percentage

## 🧠 How It Works

### Step 1️⃣

The webcam captures live video input.

### Step 2️⃣

MediaPipe detects hand landmarks in real time.

### Step 3️⃣

The system analyzes finger positions and hand movements.

### Step 4️⃣

Gestures are recognized and processed.

### Step 5️⃣

Visual effects, particles, animations, and audio feedback are generated dynamically.

### Step 6️⃣

Users interact with the experience through natural hand gestures.

##  Getting Started

### Clone the Repository

```bash
git clone https://github.com/RohithaKavala/gesturevision-hand-tracking.git
```

### Run the Project

1. Open the project folder.
2. Launch `index.html` in a modern browser.
3. Allow camera permissions.
4. Click **Enter Experience**.
5. Start interacting using your hands.
